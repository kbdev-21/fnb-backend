## Overview
A full-stack web application that supports online ordering for customers, real-time order management for store staff, and centralized administration for managing operations across multiple stores.

## Tech Stack
- **Backend:** Spring Boot  
- **Frontend:** React, Shadcn UI, Tailwind CSS  
- **Database:** PostgreSQL  
- **Storage:** Supabase Storage  

## Key Features
- Customer ordering system with store selection and promotions  
- Real-time order management for staff  
- Role-based access control (JWT authentication)  
- Flexible menu system with customizable options  
- Business analytics and reporting  

## Getting Started

### Prerequisites
- Java 17+  
- Node.js  
- PostgreSQL  

### Installation
```bash
# Clone repository
git clone https://github.com/your-username/your-repo

# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
npm install
npm run dev
