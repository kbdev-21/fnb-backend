# Multi-store F&B Management + E-commerce System (Backend repository)

## Overview
A full-stack web application that supports online ordering for customers, real-time order management for store staff, and centralized administration for managing operations across multiple stores.

## Tech Stack
- **Backend:** Spring Boot  
- **Frontend:** React, Shadcn UI, Tailwind CSS  
- **Database:** PostgreSQL  
- **Storage:** Supabase Storage
